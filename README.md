# 📄 Python URL to PDF API (Invoice & Report Generator)

A clean, fast, and simple Python script to convert any public URL or HTML page into a print-ready A4 PDF. Perfect for backend developers needing to generate invoices, receipts, and dynamic reports without managing Headless Chrome infrastructure.

## 🚀 How to use

1. **Get your API Key:** Get your free API key at [RapidAPI - URL to PDF Generator](https://rapidapi.com/titavares33/api/urltopdf-invoices-reports).

2. **Install dependencies:**
   ```bash
   pip install requests

3. Run the script:
Copy the example.py file, insert your RapidAPI key, and run:
  python example.py

⚙️ Features
- Full CSS & Layout Support: Perfect rendering of Tailwind, Bootstrap, and custom fonts.

- Background Printing: Automatically includes background colors and images (crucial for invoices).

- Network Idle Ready: Waits for all images and external assets to load before capturing the PDF.

- No Infrastructure Needed: Stop wrestling with Puppeteer, Playwright, or wkhtmltopdf on your server.
